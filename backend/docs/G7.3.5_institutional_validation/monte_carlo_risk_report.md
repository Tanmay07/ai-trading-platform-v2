# Monte Carlo Risk Report

We generated 10,000 bootstrap simulations by randomly resampling the executed trade distribution with replacement to construct confidence intervals around the expected return and drawdown.

## 90% Confidence Intervals
- **CAGR (5th Percentile - Worst Case):** 41.65%
- **CAGR (50th Percentile - Median):** 209.38%
- **CAGR (95th Percentile - Best Case):** 576.75%
- **Max Drawdown (95% Value at Risk):** -35.62%
