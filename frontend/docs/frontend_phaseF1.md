# Frontend Phase F1: AI Intelligence Workstation

## Overview
Phase F1 extends the existing AI Trading Platform frontend to fully expose the institutional-grade capabilities built in Backend Phases 1-12.
We preserved the existing user experience (Dashboard, Discovery, Full Analysis, Paper Trading, etc.) while bolting on a suite of new "AI Intelligence" screens that act as a Command Center.

## New Architecture
- **State Management**: Zustand (available for future complex state, currently using local component state for rapid deployment).
- **Styling Engine**: Upgraded to **TailwindCSS v4**, running via Vite Plugin. We configured it to seamlessly coexist with the existing vanilla CSS glassmorphism classes (`.glass-panel`, `.btn`).
- **Feature Modules**: Organized new logic into `/src/pages` and `/src/components/ui`.

## New Capabilities Added

1. **Portfolio Intelligence** (`/ai/portfolio`): 
   - Interactive drill-down into specific holdings.
   - Live AI reasoning, Kelly Allocations, and Portfolio Correlation metrics.
2. **Market Intelligence** (`/ai/market`):
   - Real-time Sector Rotation heatmaps.
   - Institutional Flow tracking (FII/DII) and global market correlations.
3. **AI Decision Center** (`/ai/decision`):
   - A visual React-style node map showing the exact architecture of the Phase 4 Multi-Agent system (Technical, Macro, Risk Agents).
   - Shows individual agent confidence scores and their localized reasoning.
4. **Research Lab** (`/ai/research`):
   - The UI for Phase 11.
   - Tracks active AI experiments, the Alpha Leaderboard, and the Feature Store registry.
5. **Strategy Center** (`/ai/strategy`):
   - Strategy Governance dashboard.
   - Allows administrators to visually promote or demote AI strategies to production and adjust their capital allocation bounds.
6. **Command Center** (`/ai/command`):
   - The ultimate C-Suite dashboard summarizing Market Data health, Model Uptime, Kubernetes Pod scaling, and Execution slippage logs. Includes the global Kill Switch.
7. **AI Insights Drawer**:
   - A persistent floating action button available on all screens. Slides open to reveal push notifications generated by the AI Supervisor (e.g. "BEL has highest breakout probability today").
8. **Weekly Targets (Upgraded)**:
   - Enhanced the existing `TargetProfitSuggestions` UI to show Portfolio Drawdowns, AI Confidence, Market Regimes, and an expandable "AI Intelligence Deep Dive" accordion for every single stock inside a recommended portfolio bundle.
